# /stock - 每日股票监控

运行A股+美股+港股每日监控系统。**Pipeline 只负责抓数据 + 检测异动(几十秒)，
异动的 AI 分析由你(Claude agent)在对话里直接完成** —— 不调用任何 LLM API，
因此用 `--dry-run` 跳过 pipeline 内置的 LLM 调用(那个免费模型极慢)。

## 执行步骤

1. 运行 pipeline（**dry-run**：抓数据 + 检测异动 + 基本面，跳过内置 LLM，约几十秒）：
```bash
cd /Volumes/SSD2T/Users/fortune/2026-投资/2026-stock-monitor
./run.sh --today --dry-run
```
> 不要去掉 `--dry-run`：带上它才不会触发那个慢速免费 LLM；分析交给你来做。

2. 如果 pipeline 失败，检查错误原因：
   - AKShare 获取A股失败 → 可能限流，稍后重试或检查网络
   - 美股/港股获取失败 → 检查网络 / `FINNHUB_API_KEY`（美股今日报价，可选；Yahoo 仍能回填历史）
   - 数据全空 → 检查 `.venv` 与网络

3. 读取 pipeline 生成的数据报告：
```bash
cat output/$(date +%Y-%m-%d).md
```
   报告结构（自上而下）：
   - **⚠️ 数据质量警告**（若有）：如「美股数据疑似未更新」→ 当日忽略美股行情/异动
   - **市场体温计**：指数 + 隔夜外围(纳指/费半/恒指) + 涨跌/涨跌停家数 + 沪市两融
     + **Regime 状态机**（进攻/换挡/震荡/熄火/杀跌）+ 近5日轨迹
     → **regime 是数据算出来的每日必判基准，直接采用，只在有强反证时才推翻**；
       连续降档（如 熄火→杀跌）本身就是最强风险信号
   - **板块聚合信号**：各板块 涨/跌只数 + 中位涨跌幅 + 异动数 → 直接判「板块效应 vs 单票行为」
   - 市场概览 / A股基本面 / 美股港股基本面
   - **异常波动信号**（同股已合并，severity 已按个股波动率 z-score 校准）
   - **主力资金流向**（watchlist A股 Top/Bottom10）+ **龙虎榜命中**（游资定性铁证）
   - **未来风险日历**：美股财报 / A股解禁 / 新股申购 → **事前预警，分析里必须引用**

4. **昨日判断验证（先复盘再分析，防止重复犯错）**：
   - 读取最近一份 `output/judgments/*.json`（按文件名取最新）
   - 逐条对照今日数据判定：`确认 / 证伪 / 待定`
   - 在 analysis.md 开头写「昨日判断验证」小节；**连续证伪同方向判断 = 市场无主线的确认信号**

5. **Claude 直接分析异动（核心步骤，代替内置 LLM）**：
   - 从「异常波动信号」表读取全部异动；**预测性信号优先分析**（见下表），它们先于价格崩溃：

   | 信号 | 含义 | 框架对应操作 |
   |------|------|-------------|
   | distribution_warning | 高位放量滞涨 / 缩量创新高(UTAD) | 派发预警 → 减仓预警 |
   | volume_dryup | 深跌后连续地量 | Markdown尾声候选 → 左侧观察名单 |
   | lone_strength | 板块普跌单票逆势极强 | 独苗背离 → 补跌风险，别追（实证：新易盛07-14→07-18） |
   | momentum_exhaustion | 连涨但量逐日递减 | Effort递减 → 动能衰竭，不追高 |

   - 结合基本面表 + **主力资金流**（异动方向与主力资金背离 = 可疑）+ **龙虎榜**（上榜 = 游资票定性）
   - **应用「多空逻辑 × 威科夫」分析框架**，判断主线/非主线、量价结构阶段；regime 直接引用体温计
   - 对每个 **high** 异动写简明分析：①可能原因 ②短期风险/机会 ③关注信号
   - **medium** 异动汇总成表，逐行一句话点评
   - 引用「未来风险日历」：财报/解禁临近的持仓标的必须提示
   - 把分析写入 `output/$(date +%Y-%m-%d)-analysis.md`（标题「🤖 Claude 异动分析 -- 日期」），并在对话中给出要点

6. **写入今日判断存档（复盘闭环）**：
   - 保存 `output/judgments/$(date +%Y-%m-%d).json`，schema：
   ```json
   {"date": "YYYY-MM-DD", "regime": "杀跌",
    "judgments": [{"id": "j1", "claim": "有色换挡成立",
                   "falsifiable": "明日有色板块中位数转跌或龙头缩量破位",
                   "horizon_days": 2, "status": "open"}]}
   ```
   - 每条判断必须带**可证伪条件**和期限；只记关键判断（3-6条），不记显然事实
   - 同步维护 `output/focus_list.json`（重点关注清单），记录进出与理由

7. (可选) WebSearch 补充新闻：对重点异动股票搜索 `'代码 A股/港股 最新消息'` 或 `'TICKER stock news today'`，把关键信息融进你的分析。

8. (可选) 实时行情：`python -m src.realtime --watchlist watchlist.md`

9. 汇总输出：
   - **Regime + 近5日轨迹**（降档必须醒目提示）
   - 各市场数据获取情况（A股/美股/港股 数量）+ 数据质量警告
   - 异动数量；high/medium 分布；预测性信号单独列出
   - 昨日判断验证结果（确认/证伪各几条）
   - Claude 分析要点 + analysis 文件路径 + 数据报告路径
   - 未来风险日历要点（3日内的财报/解禁）

## 多空逻辑 × 威科夫 分析框架

> 源自 `/Volumes/SSD2T/Users/fortune/2026-投资/多空逻辑/` 中 Jerry Ma 的股票文章
> （核心篇目：对股市的理解与交易方法的思考、A股周期论、同样的牛市不同的泡沫、
> 关于股市的三点争议、在自己的时代赚PE在下一个时代赚EPS、A股的未来指数型个股的牛市），
> 融合威科夫(Wyckoff)量价理论。分析异动时按此框架定性。

### 1. 结构性牛市观（每日必判）

**指数平稳走牛 = A主线涨到怀疑人生 + B凑热闹瞎震荡 + C非主线被抽资金阴跌。**
- 每次分析先给股票归类：A主线 / B打酱油 / C失血。抓不到主线 = 牛市与你无关
- 牛市不止一个主线，主线轮流发力顶指数；**主线大跌时先判断是「换挡」还是「熄火」**：
  大金融护盘 + 主线内部资金流转（如光模块→半导体设备）= 换挡（科技内部轮动）；
  全板块无差别下跌 + 无护盘 = 风险偏好整体下降，资金是**离场蒸发**而非搬家
- **科技调整后资金不会流向消费/中药等"老登"板块**——低估值从来不是上涨的原因，
  景气度+叙事才是。别在调仓窗口站错队

### 2. 主线识别三要素

**事件驱动 + 板块效应 + 持续性**
- 事件驱动：政策利多/海外订单/周期拐点/业绩变化 → 改变预期、提升风险偏好
- 板块效应：不是单票涨，而是板块内多票涨停、其余大涨，且持续多日
- 持续性 = 业绩兑现的**确定性 × 速度**：业绩确定+利润释放快 = 机构票（主线，容量票，
  跌了有承接）；业绩不确定 = 游资票（一波流，情绪票，跌停板出货）
- **龙头 = 板块内涨幅最大 + 成交额最大**（都涨停则比成交额）

### 3. 威科夫量价四阶段定位（对 high 异动逐一定位）

| 阶段 | 量价特征 | 对应操作倾向 |
|------|---------|-------------|
| 吸筹 Accumulation | 低位缩量横盘，下探缩量(Spring/测试)，突破放量(SOS) | 埋伏区，分批低吸 |
| 拉升 Markup | 上攻放量、回踩缩量、沿均线有序上行 | 持有/回踩均线加仓，加速不追 |
| 派发 Distribution | **高位放量滞涨/缩量创新高(UTAD)、上冲乏力** | 减仓预警 |
| 下跌 Markdown | 放量破位(SOW)、反弹缩量、高点降低 | 回避，别接飞刀 |

- 核心是 **Effort vs Result**：量增价不涨=顶部背离；量缩价稳=抛压衰竭
- 多空逻辑原文对应："**容量票每天稳定放量上涨=市场健康；高位明显缩量滞涨=至少暂时性调整**"
- 结合 pipeline 数据落地：`量比`(>1.5放量/<0.7缩量) + `换手率` + `连续涨跌天数` + 涨跌幅

### 4. 买卖点纪律

- **买点三要素：逻辑 + 业绩 + 买点。绝不在离买点太远的地方追涨**（多数人亏钱不是逻辑错，
  是买点太差、被价格调动情绪追进去）
- 逻辑硬+业绩确定+释放快的票：斜率变陡前随便买，回踩均线分批买，**加速后不追**
- 逻辑硬+业绩不确定的票：只在启动一两根K线内买，错过就放弃
- 左侧量化买点（反转型）：**杀完估值+杀完业绩后，连续两个季度业绩同比两位数正增长 = 最佳买点**
  （第一个季度正增长的反弹不持续，第二个季度确认后低点抬高、放量上涨）

### 5. 周期宿命与禁忌

- **A股一切股票都是周期股**，没有穿越牛熊；上轮牛市的大牛股 = 下轮周期的垃圾股
- **别接盘上个时代被炒高的资产**（白酒/中药/消费茅），继承可以，接盘不行；
  创新药 > 中药；医药优选卖铲子的 CRO/CDMO/生命科学工具，不碰下游创新药企
- **在自己的时代赚PE（景气度+叙事拉估值），在下一个时代赚EPS**；
  股价高点永远对应估值高点而非业绩高点——景气度见顶/叙事改变时，业绩再好也挡不住杀估值
- AI 牛市最大风险 = **CapEx 剪刀差 + 负反馈**（铲子比金子贵→下游削减资本开支→铲子股戴维斯双杀），
  持续跟踪下游 CapEx 指引与"AI收入/GPU投入"比值叙事

### 6. 输出要求（写入 analysis.md）

- 给每个 high 异动标注：主线归类(A/B/C) + 威科夫阶段 + 是否可操作(埋伏/持有/回避/减仓)
- 周期性维护「重点关注清单」：主线龙头(容量票) + 处于吸筹/拉升早期 + 业绩确定释放快，
  剔除进入派发/下跌阶段的票
- 新闻/研报高频股：WebSearch 检索当日热门研报与新闻高频标的，与 watchlist 交叉比对
